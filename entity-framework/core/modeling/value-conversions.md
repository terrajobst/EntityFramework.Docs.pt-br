## <a name="configuring-a-value-converter"></a><span data-ttu-id="636c4-115">Configurando um conversor de valor</span><span class="sxs-lookup"><span data-stu-id="636c4-115">Configuring a value converter</span></span>

<span data-ttu-id="636c4-116">Conversões de valor são definidas nas propriedades no `OnModelCreating` do seu `DbContext`.</span><span class="sxs-lookup"><span data-stu-id="636c4-116">Value conversions are defined on properties in the `OnModelCreating` of your `DbContext`.</span></span> <span data-ttu-id="636c4-117">Por exemplo, considere uma enumeração e um tipo de entidade definidos como:</span><span class="sxs-lookup"><span data-stu-id="636c4-117">For example, consider an enum and entity type defined as:</span></span>

``` csharp
public class Rider
{
    public int Id { get; set; }
    public EquineBeast Mount { get; set; }
}

public enum EquineBeast
{
    Donkey,
    Mule,
    Horse,
    Unicorn
}
```

<span data-ttu-id="636c4-118">Em seguida, as conversões podem ser definidas no `OnModelCreating` para armazenar os valores de enumeração como cadeias de caracteres (por exemplo, "Donkey", "Mule",...) no banco de dados:</span><span class="sxs-lookup"><span data-stu-id="636c4-118">Then conversions can be defined in `OnModelCreating` to store the enum values as strings (for example, "Donkey", "Mule", ...) in the database:</span></span>

``` csharp

{
    modelBuilder
        .Entity<Rider>()
        .Property(e => e.Mount)
        .HasConversion(
            v => v.ToString(),
            v => (EquineBeast)Enum.Parse(typeof(EquineBeast), v));
}
```

> [!NOTE]  
> <span data-ttu-id="636c4-119">Um valor `null` nunca será passado para um conversor de valor.</span><span class="sxs-lookup"><span data-stu-id="636c4-119">A `null` value will never be passed to a value converter.</span></span> <span data-ttu-id="636c4-120">Isso torna a implementação de conversões mais fácil e permite que elas sejam compartilhadas entre Propriedades anuláveis e não anuláveis.</span><span class="sxs-lookup"><span data-stu-id="636c4-120">This makes the implementation of conversions easier and allows them to be shared amongst nullable and non-nullable properties.</span></span>

## <a name="the-valueconverter-class"></a><span data-ttu-id="636c4-121">A classe ValueConverter</span><span class="sxs-lookup"><span data-stu-id="636c4-121">The ValueConverter class</span></span>

<span data-ttu-id="636c4-125">Isso pode ser útil quando várias propriedades usam a mesma conversão.</span><span class="sxs-lookup"><span data-stu-id="636c4-125">This can be useful when multiple properties use the same conversion.</span></span>
